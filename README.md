# Personal Portfolio Website

## Project Overview
A full-stack personal portfolio website built to showcase my projects, skills, and experience. The website integrates frontend, backend, and database functionality to provide a dynamic and interactive portfolio.

## Key Features
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js with Express.js
- **Database:** MongoDB (local or Atlas)
- **Projects Section:** Displays all portfolio projects with descriptions, technologies used, and GitHub links
- **Contact Section:** Connect via LinkedIn
- **Responsive Design:** Works on desktop and mobile

- Personal Portfolio Website
